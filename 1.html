HTML

HTML은 웹 페이지 및 응용 프로그램을 만드는 데 널리 사용되는 표준 마크업 언어입니다. 웹 브라우저가 서버나 로컬 저장소에서 HTML 문서를 받으면 HTML 코드를 사용하여 멀티미디어 웹 페이지를 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 설명하는 데 사용되며 원래는 문서의 모양에 대한 단서를 포함했습니다.

HTML 요소는 HTML 페이지의 빌딩 블록 역할을 합니다. HTML 구성을 사용하여 개발자는 이미지 및 대화형 양식과 같은 다양한 개체를 렌더링된 페이지에 포함할 수 있습니다. HTML은 제목, 단락, 목록, 링크, 인용 부호 및 기타 요소를 포함하여 텍스트의 구조적 의미를 표시하여 구조화된 문서를 만드는 방법도 제공합니다. HTML 요소는 꺾쇠 괄호를 사용하여 작성된 태그로 표시됩니다.
