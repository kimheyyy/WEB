JavaScript

JavaScript(JS로 약칭)는 고급, 동적, 프로토타입 기반, 다중 패러다임 및 해석 프로그래밍 언어입니다. 약한 유형입니다. 즉, 데이터 유형이 명시적으로 선언되지 않으며 구문이 유연하고 표현적인 코딩을 허용합니다. HTML 및 CSS와 함께 JavaScript는 World Wide Web 콘텐츠 제작에 사용되는 세 가지 필수 기술 중 하나입니다. JavaScript는 웹 페이지를 대화식으로 만들고 비디오 게임을 포함한 온라인 프로그램을 만드는 데 널리 사용됩니다. 대부분의 웹 사이트는 JavaScript를 사용하며 모든 최신 웹 브라우저는 내장 JavaScript 엔진을 통해 플러그인 없이 JavaScript를 지원합니다. ECMAScript 사양을 기반으로 하는 언어의 서로 다른 구현을 나타내는 JavaScript 엔진이 많이 있습니다. 일부 엔진은 사양을 완전히 지원하지 않지만 많은 엔진이 ECMA 표준 이외의 추가 기능도 제공합니다.
